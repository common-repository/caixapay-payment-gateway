== Description ==

Now accept [Caixapay](https://Caixapay.com) cxp as payment currency with the first CaixaPay payment gateway!


== Main features ==

 - powered by [Caixapay for Merchants](https://caixapay-api.com/ "Caixapay for Merchants" ) payment gateway,
 - **multi-currency** (CXP, BTC, EUR, USD, AUD, BRL, CAD, CHF, CLP, CNY, DKK, GBP, HKD, INR, ISK, JPY, KRW, NZD, PLN, RUB, SEK, SGD, THB, TWD),
 - **anonymous** : no api key needed,
 - extremely **simplified setup**,
 - includes a Woocommerce integrated optional **debugging tool**,
 - other Woocommerce **gateways compliant**.

== Installation ==

= Minimum Requirements =

* PHP version 5.2.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* Woocommerce version 3.0


= Manual installation =

- download woocxp-gateway.zip file

- from your wordpress site admin dashboard :
    1) plugin / add new / upload plugin
    2) activate the plugin
    3) go to 'Woocxp / settings' page to set up and enable Caixapay cxp payments

That's all !
